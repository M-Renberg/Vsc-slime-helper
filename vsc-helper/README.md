# Slime Helper 

Your personal coding buddy and rubber duck!

Slime Helper brings a living, breathing anime style slime to your desktop that reacts to how you code.

## Features

* **Error and Warning checking:** The Slime reacts to the code.
* **Streaks:** Keeps track of your coding flow.
* **Health Check:** Reminds you to take breaks, drink water, and sleep.
* **Git tracking:** keeps track so you don't forget to commit and push your code.
* **Commentary:** Slime helper adds colorful commentary on your code while you're coding.

## Requirements 
* Windows OS (since it uses a WPF companion app).

## Setup 
The extension comes with the Slime Helper app included. 
1. Install the extension.
2. Run command `Start Slime Helper` if auto start don't work.
3. Enjoy your new coding buddy!

### Start Slime:
Press Ctrl + Shift + P (Command Palette).

Write: Start Slime Helper.

Press Enter.
